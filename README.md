USB-C to UART Adapter Board (CH340)

A compact USB Type-C to UART Adapter Board designed using the CH340 USB-to-Serial converter IC. This board provides a reliable USB-to-TTL serial interface for programming, debugging, and communicating with microcontrollers and embedded systems.

**Features**

    USB Type-C interface for modern connectivity
  
    CH340 USB-to-UART bridge IC
  
    Supports 3.3V and 5V UART communication
  
    TX, RX, VCC, and GND header pins
  
    Power and communication status LEDs
  
    Compact and easy-to-use PCB layout
  
    Designed using KiCad

**Applications**

    STM32 programming and debugging
  
    Arduino and ESP32 serial communication
  
    Bootloader flashing
  
    Embedded system development
  
    Serial console and debugging

**PCB Design Highlights**

      Schematic capture in KiCad
    
      Professional PCB layout
      
      Proper USB differential pair routing
      
      Decoupling capacitors placed near the IC
      
      Ground plane for improved signal integrity
      
      Design Rule Check (DRC) and Electrical Rule Check (ERC) verified
      
**Learning Outcomes****
      
      USB interface design
      
      USB-to-UART communication using CH340
      
      PCB component placement and routing
      
      Power distribution and decoupling techniques
      
      Hardware debugging and PCB design workflow
  
**  Tools Used**
  
      KiCad
      
      CH340 USB-to-UART IC
      
      USB Type-C Connector

This project demonstrates the design of a reliable USB-C to UART adapter suitable for embedded development, firmware flashing, and serial debugging.
